Super Spaghetti

Description

This is a description of a tasty spaghetti squash that should be appropriately sized. It is unclear which really the size should be appropriate - this text or the spaghetti?

Ingredients

  • Frozen spaghetti (store bought)
  • Heat (microwave will suffice)
  • Plate

Steps

  1. Unwrap frozen spaghetti.
  2. Place spaghetti in microwave or other heat source.
  3. Place heated spaghetti on selected plate.
